# Convention de nommage

## MODEL
> Nom des méthodes avec préfixes en :

  + insert*
  + select*
  + update*
  + delete*

## CONTROLLER
> Nom des méthodes avec préfixes en :

  + create* (pages d'ajout, MODEL : insert*)
  + get* (MODEL : select*)
  + edit* (pages d'édition)
  + refresh* (MODEL : update*)
  + remove* (MODEL : delete)

## VIEW
> Nom des vues avec préfixe en :

  + create* (pages d'ajout, MODEL : insert*)
  + get* (MODEL : select*)
  + edit* (pages d'édition)
  + refresh* (MODEL : update*)
  + remove* (MODEL : delete)
